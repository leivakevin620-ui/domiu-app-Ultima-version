# Controles de cumplimiento

- Minimización de datos del invitado.
- Separación de notas por audiencia.
- Trazabilidad de actor y motivo.
- Conservación de snapshots del hecho comercial.
- No almacenamiento de credenciales de pago.
- Privilegios server-side y RLS.
- Retención de borradores limitada.
- Logs sin secretos.
