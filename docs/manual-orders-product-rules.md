# Reglas de producto

- Confirmación explícita obligatoria.
- Pedido manual visible como tal.
- Tarifa de domicilio a cargo del cliente.
- Invitado sin cuenta.
- Pickup sin repartidor.
- Ajustes y excepciones con motivo.
- Borrador sin impacto operativo.
- Misma experiencia y dominio para admin/comercio.
