# Modelo de amenazas

Activos: pedidos, inventario, precios, datos de clientes, notas internas, asignación y registros financieros.

Fronteras: navegador, APIs Next.js, cliente de servicio Supabase, función PostgreSQL y triggers.

Controles principales: autenticación remota, rol, tenant, same-origin, validación estricta, consultas parametrizadas, bloqueo de filas, idempotencia, RLS, mínimos privilegios y auditoría.

Riesgo residual: un administrador legítimo puede ejercer excepciones autorizadas. Cada excepción exige motivo y queda auditada.
