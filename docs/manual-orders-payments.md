# Pagos en pedidos manuales

Métodos admitidos por el dominio actual: efectivo, transferencia, tarjeta crédito, tarjeta débito y billetera.

Crear un pedido no marca el pago como completado por defecto. Transferencias y pagos no efectivos completados requieren referencia externa. Los datos se integran con los triggers existentes de transacciones y liquidaciones.

La tarifa de domicilio se cobra al cliente. El desglose financiero calcula producto, descuento, recargo, propina, domicilio, tarifa de servicio, ganancias del negocio, repartidor y plataforma.

No se almacenan PAN, CVV, PIN, credenciales bancarias o tokens de proveedores.
