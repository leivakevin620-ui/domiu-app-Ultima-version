# Protección del despliegue

El despliegue reutiliza el núcleo financiero existente de Supabase y no vuelve a crear tablas, saldos ni jornadas. La configuración monetaria se actualizó mediante una migración independiente y auditable.

No se borran pedidos, usuarios, comercios, repartidores, movimientos financieros ni liquidaciones anteriores.
