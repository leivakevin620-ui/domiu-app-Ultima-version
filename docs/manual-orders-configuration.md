# Configuración funcional

Campos de `businesses`:

- `allow_custom_manual_products`: permite al comercio añadir artículos que no existen en catálogo.
- `allow_manual_delivery_fee_override`: permite al comercio ingresar tarifa manual con motivo.

Administración puede usar ambas capacidades por rol. Mantenerlas desactivadas por defecto para nuevos negocios.

La expiración actual de borradores es 30 días. Cambiarla requiere una migración o un proceso de limpieza que respete borradores activos.
