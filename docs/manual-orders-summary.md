# Resumen ejecutivo

Pedidos manuales permite que administración y negocios conviertan solicitudes externas en pedidos completos dentro del flujo normal de DomiU. El sistema admite invitados sin autenticación, valida catálogo y stock, calcula valores en backend, conserva snapshots, evita duplicados y audita excepciones.

La implementación reemplaza el flujo legado de “solo domicilio” y evita crear cuentas artificiales para invitados.
