# Despliegue de producción — DomiU App Última Versión

Proyecto Vercel: `domiu-app-ultima-version`

Rama de producción: `master`

Este despliegue incorpora:

- creación manual de domicilios para administración y negocios;
- formulario con extracción de mensajes de WhatsApp;
- cálculo de ruta, tiempo y tarifa;
- asignación o publicación para repartidores;
- mejora de contraste y legibilidad;
- diseño visual premium;
- corrección de proporciones y ubicación del logo oficial.

La aplicación operativa anterior utiliza la rama `main` y queda fuera de este despliegue.
