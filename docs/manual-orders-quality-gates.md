# Puertas de calidad

Un cambio no se fusiona cuando falla alguno de estos controles:

- secretos o `.env` rastreados;
- vulnerabilidad alta o crítica;
- contratos Zod;
- pruebas de arquitectura y seguridad;
- TypeScript;
- build de Next.js;
- reconstrucción de migraciones;
- Vercel Preview;
- logs de ejecución.
